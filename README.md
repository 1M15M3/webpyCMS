webpyCMS0.1
========

webpyCMS is a fastest and smallest CMS base on web.py


webpyCMS是基于web.py的网站内容管理系统，小巧实用，非常适合做博客和个人简历网站。

当前版本为0.1“概念版”，搭好了基本框架，简单实现了管理文章、管理评论的功能。

没有demo站点，前台样式参照http://new.geekgo.net
